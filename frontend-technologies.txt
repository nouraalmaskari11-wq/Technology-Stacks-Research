Frontend Technologies (frontend-technologies.txt)

These are used to build the user interface and handle the client-side logic:

HTML – Structure of web pages.

CSS – Styling and layout of web pages.

JavaScript (JS) – Adds interactivity and dynamic behavior to web pages.

Additional popular frontend tools/frameworks (optional for research):

React.js, Angular, Vue.js

Tailwind CSS, Bootstrap

React.js – Component-based UI library for building interactive web apps; widely used in modern web development.

Vue.js – Progressive framework for building user interfaces; flexible and easy to integrate.

Other notes: Both React and Vue handle the view layer but rely on CSS/HTML for styling and structure.
